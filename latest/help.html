Matisse

Collaborative WhiteBoard

Help

What is Matisse

Matisse is a collaborative whiteboard. Create a whiteboard or open your existing one, send the url to your friend and then work on it together. All your changes are visible to them and their changes to you.

Key shortcuts

  • "Delete" for deleting a shape
  • "Shift + Arrow" for big moves of the selected shape
  • "Arrow" for finer moves of the selected shape

How to change the properties

Select any element on the canvas, a small toolbox would appear on top of it. Click the gear icon, it would open the properties box.

What does Show Image do

Show image opens the contents of the canvas as a new image. Save the image locally using "save image as" from the browser context menu. Currently it does not support image elements embedded in the whiteboard

Future Direction
